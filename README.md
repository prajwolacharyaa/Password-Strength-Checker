# Password Strength Checker

A Flask-based password strength checker project for a cybersecurity student portfolio.

## Features

- Real-time password analysis
- Length and complexity checks
- Entropy calculation
- Common pattern detection
- Common password wordlist check
- Clean cyber-themed UI

## Project Structure

- `app.py` - Flask app
- `checker/entropy.py` - entropy calculation
- `checker/patterns.py` - repeated/sequential pattern checks
- `checker/wordlist.py` - common password list loader
- `checker/strength.py` - main scoring logic
- `templates/index.html` - frontend template
- `static/style.css` - styles
